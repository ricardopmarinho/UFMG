Ambiente computacional aonde os TPs foram deselvolvidos: Windows, IDE utilizada Dev-C++
Instruções para ambos os arquivos:
- Favor não colocar um caracter na entrada! Caso o usuário entre com caracteres o programa entrara em um loop infinito.
- Tanto na soma maxima quanto no quadrado mágico, se o usuário inserir uma entrada fora dos intervalos solicitados, uma mensagem de erro aparecerá e espera-se a entrada de um número válido.
- O programa rodará apenas uma vez gerando a saída solicitada pela entrada.

Soma Máxima:
- Se todos os números do vetor forem negativos a soma deve ser 0, devido a isso, a mensagem "Indice invalido" aparecerá.
- Analisando um vetor como 1, 0, 0 os indices considerados para a maior soma são de 1 a 3 (isso acontece pela condição de serem todos positivos). Já que não foi especificado se o conjunto considerado deverá ser o menor subconjunto possível.
- Analisando um vetor como 1, 0, 0, -1 os indices considerados são de 1 a 1. 

Quadrado Mágico:
- O programa gera apenas um quadrado mágico.
