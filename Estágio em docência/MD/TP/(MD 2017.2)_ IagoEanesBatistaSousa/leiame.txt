Observação:
Os programas foram escritos na linguagem de programação C, no ambiente Linux
(Kernel v 4.13.9) em editor de texto simples. Foram compilados com auxílio do
GCC (v 7.2.0).

                                Soma Maxima
Para este problema, não foram utilizadas diretrizes nem bibliotecas específicas:
Basta compilar o programa fonte com auxílio do comando abaixo, no emulador de terminal:
(é importante recordar que a compilação deve ser feita no mesmo diretório em que se encontra o arquivo fonte)
$: gcc somamax.c -o somamax
Após a compilação, ainda no emulador de terminal, basta executar o programa compilado, com a seguinte linha de código:
$: ./somamax

                                  Quadrado Mágico
Para este problema foi utilizada a função pow (que realiza potenciação), que está contida na biblioteca math.h.
Para compilação no GCC é necessário passar o parâmetro que referencia essa biblioteca durante a compilação
Basta compilar o programa fonte com auxílio do comando abaixo, no emulador de terminal:
(é importante recordar que a compilação deve ser feita no mesmo diretório em que se encontra o arquivo fonte)
$: gcc qmagico.c -o somamax -lm
Após a compilação, ainda no emulador de terminal, basta executar o programa compilado, com a seguinte linha de código:
$: ./qmagico
